# Kiva-ClientRobot

모바일로 물건의 위치확인후 움직일 장소를 선택해주면 움직이는 로봇



## About the Project

- OpenCV 이용해서 QR코드로 위치확인
- 바퀴를 이용해서 움직이는 로봇
- 상단에 모터를 이용해서 선반을 들수있도록 제작
- 최단거리 알고리즘 적용해서 최단거리 계산



## Project status

개발 기간 부족으로 모터파트는 미개발



## Getting Started

### Requirements

- Nvidia Jetson Tx1
- Ubuntu 14.04 LTS
- OpenCV
